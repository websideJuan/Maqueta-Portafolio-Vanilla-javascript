Historia del lenguaje HTML

HTML es un lenguaje de marcado que permite crear páginas web mediante etiquetas que definen la estructura y el contenido de un documento. HTML es un estándar que se actualiza periódicamente para adaptarse a las necesidades de los desarrolladores y los usuarios.
